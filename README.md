# chooser

Ein kleines Tool, mit dem man einfach zwischen verschiedenen Optionen wählen kann. Inspiriert an dem deutschen Wahl-o-mat. Alle Optionen und deren Eigenschaften können in der config.json eingestellt werden.

Die momentane config.json stellt verschiedene Linux Distros zur Auswahl und kann [hier](https://xxxb.codeberg.page/chooser/) ausprobiert werden.
